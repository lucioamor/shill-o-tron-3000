Shill-O-Tron 3000 - Permissions Explained

  • activeTab: Allows the extension to analyze the Reddit thread you’re currently viewing to detect potential shill activity.
  • identity: Enables secure Reddit OAuth authentication, allowing the extension to fetch your comment history with your consent for shill detection.
  • storage: Permits the extension to store your Reddit access token locally in Chrome’s storage, ensuring seamless API access without repeated logins.
  • Host permissions (https://www.reddit.com/*, https://oauth.reddit.com/*): Required to access Reddit’s API, used solely to retrieve public comment data for analyzing link patterns. No personal data is shared or stored beyond this purpose.

For more details, see the Legal Disclaimer and Privacy Policy. This extension complies with Chrome Web Store and Reddit policies, using permissions only for its experimental, good-faith purpose.
