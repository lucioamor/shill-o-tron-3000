Shill-O-Tron 3000 - Privacy Policy

Welcome to the Privacy Policy for Shill-O-Tron 3000, a proof-of-concept Chrome extension designed to zap sneaky shillers on Reddit! This policy explains how we handle data while enhancing your Reddit experience in a humorous, experimental way. We value your privacy and comply with the Chrome Web Store Program Policies and Reddit User Agreement.

Data We Collect

We collect the following data to provide our shill detection functionality:

  • Authentication Data: A Reddit access token is collected via OAuth with your consent. This token is stored locally in Chrome’s storage and used to fetch comment data from Reddit’s API. We do not access passwords or other credentials.
  • Website Content: Public comment data from Reddit threads (e.g., text, hyperlinks) is temporarily processed in real-time to detect repetitive link promotion. This data is not stored beyond your session.

We do not collect personal identification information (e.g., name, email), health data, financial data, personal communications, location, web history, or user activity beyond Reddit comments.

How We Use Data

Data is used solely for the extension’s single purpose: detecting potential shillers by analyzing link patterns in Reddit comments. The access token enables API requests, and comment data is processed to flag suspicious activity with a playful interface. No data is used for purposes unrelated to this goal, such as determining credibility or lending.

Data Sharing

We do not sell, transfer, or share your data with third parties, except as required by law or Reddit’s API terms (e.g., during OAuth authentication). All data processing occurs locally on your device, ensuring your privacy.

Transparency and Compliance

Our algorithm is fully transparent, with source code available on GitHub. We only use permissions necessary for our experimental purpose (e.g., Reddit API access via OAuth). See the Permissions Explained for details.

Your Choices

By installing Shill-O-Tron 3000, you consent to this data usage. You can revoke access by uninstalling the extension, which removes the locally stored access token. No data persists after uninstallation.
